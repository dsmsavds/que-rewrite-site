Privacy Policy

What data Que collects?

· The server keeps records of user IDs and server IDs.
· Warned members are also tracked and recorded by their corresponding member IDs.
· The server configuration settings for the Que system are saved.
· While messages themselves are not logged, they can be examined if the server has an antiflood or antilink system enabled.

Why does Que collect this data?

· To backup Que's configuration files.
· To log all moderation changes made in every server before they are implemented.
· For tracking purposes, to record relevant information.
